# FairPreferentialChoiceAlgorithms

**Test system for the evaluation of course allocation procedures**  
by **Dennis Strohbach**, 2025

---

## Project description

This project offers a modular test system for analyzing and evaluating allocation procedures for preference choices, especially in the context of course choices.  
The aim is to objectively compare different algorithms on the basis of fairness metrics.
